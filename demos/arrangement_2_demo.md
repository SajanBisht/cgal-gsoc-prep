# Arrangement_2 Demo

## Overview

Built and executed CGAL Arrangement_2 example to understand core functionality.

## What I Tested

- Constructed arrangement with segments
- Explored vertices, edges, and faces
- Observed traversal using halfedges

## Observations

- Arrangement structure is based on DCEL (Doubly Connected Edge List)
- Vertex, Halfedge, and Face bindings directly map to this structure
- Traversal APIs in bindings correspond to circulators/iterators in CGAL
